# Agents

Reusable, model-neutral agent definitions live here. They are written as plain
Markdown so they can be used by Codex, Claude, Gemini, Cursor, or any other LLM
assistant without tool-specific syntax.

## Available agents

| Agent | Purpose |
| --- | --- |
| [Flow Sentinel](flow-sentinel/AGENT.md) | Reviews requested changes for product-flow impact, scope drift, reliability risk, and unwanted load or cost before implementation. |
