# Domain Dictionary

This document establishes the ubiquitous language used throughout the project.

## Terms
*   **Organizer:** The entity (company or individual) hosting an event.
*   **Attendee:** A registered participant of an event.
*   **Connection:** A confirmed relationship between two users (part of Event Memory).
*   **Lead:** A user who has shown measurable interest in an Exhibitor or Sponsor.
*   **Event Memory:** The persistent, cross-event graph of relationships and interactions.
