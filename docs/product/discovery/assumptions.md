## Core Assumptions

### 1. The Post-Event Value Assumption
*   **Assumption:** Attendees and organizers want to use an event-specific platform *after* the event concludes.
*   **Risk if Invalid:** "Event Memory" features will go unused, and value proposition drops significantly.
